# Abfallen & Aufmarschieren — animierte Erklärung

Eine animierte, interaktive Erklärung der Marschformen **Abfallen** und **Aufmarschieren** für Blasmusikantinnen und Musikanten — speziell für die, die keine Stabführer-Ausbildung haben.

🌐 **Live ansehen:** https://eggerjared.github.io/marschieren/

## Was ist drin?

- Drei Varianten nach offiziellem Wiki-Artikel des Österreichischen Blasmusikverbandes:
  1. **Reißverschluss-Methode** — Flügelmusiker rücken Linie für Linie ein
  2. **Gleichzeitig (still)** — alle Flügelmusiker zugleich, ohne akustisches Zeichen
  3. **Mit Trommel-Signal** — zusätzliche akustische Signale der Großen Trommel
- Animierte SVG-Formation mit Stabführer, Großer Trommel, Flügelmusikern, Mittel-Reihen
- Live-Zeitachse mit Takten und Zählzeiten
- Glossar-Tooltips für alle Fachbegriffe
- Rollenfilter („Was muss ich tun, wenn ich Flügelmusiker bin?")
- Quiz pro Variante zur Selbstkontrolle
